# XLocale

一款基于 SwiftUI 开发的 macOS 本地化工具，致力于简化 Xcode 项目的国际化流程。

## ✨ 功能特点

- 🌍 支持中文和英文双语界面
- 📱 导入导出 Localizable.strings 文件
- 🔄 自动同步缺失的翻译键值
- 🎯 智能检测未使用的本地化字符串
- 🤖 集成 OpenAI 辅助翻译
- 🎨 原生 macOS 设计风格
- 🌓 自适应明暗主题

## 💻 系统要求

- macOS 13.0 (Ventura) 或更高版本
- 支持 Apple Silicon 和 Intel 芯片

## 🛠 技术栈

- SwiftUI
- OpenAI
- SwiftUIX
- SwifterSwift
- Tuist

## 🔑 使用说明

1. 打开应用后，选择 Xcode 项目所在目录
2. 应用会自动扫描并列出所有的本地化文件
3. 在界面中直接编辑翻译内容
4. 可选择使用 AI 辅助翻译功能
5. 修改完成后自动同步到项目文件中
